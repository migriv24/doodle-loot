# Doodle Loot

A Doodle Jump-style mobile platformer built entirely with AI-written code.

---

## This game was written by AI

Every line of code in this repository was generated by [Claude](https://claude.ai) (Anthropic's AI). No human wrote any of the game logic, rendering, physics, or build tooling. This project exists as a proof of concept that AI can design, architect, and implement a fully functional mobile game from scratch — including Android APK packaging.

The human's role was to describe features and give feedback. The AI's role was to build everything.

---

## What it is

A vertical platformer where you bounce upward through procedurally generated platforms, fight enemies and bosses, earn money, and spend it in a shop. Features include:

- Platform rarity system (Common → Legendary) with monetary values
- 7 power-ups: Rocket Boost, Tap Platform, Slow-Mo, Shield, Coin Magnet, Multi-Shot, Spring Boots
- Skill tree for character upgrades
- Boss battles at section breaks with roguelike upgrade choices after each boss
- Shop with 7 unlockable skins and loot boxes
- Frame-perfect jump mechanic (7-frame window for 1.5x bounce)
- Particle effects, parallax starfield backgrounds, section-themed colors
- Runs on desktop (Windows/Mac/Linux) and Android

## Architecture

Hexagonal (ports & adapters) architecture — the game logic in `game/` has zero pygame dependency. Swapping to a different renderer or platform only requires a new adapter.

```
game/          # Pure Python game logic — no framework dependency
  entities/    # Player, Platform, Enemy, Boss, Projectile, Particle
  systems/     # Physics, Generation, Combat, Economy, Progression
  world.py     # Main game state machine
adapters/
  pygame_adapter/  # Desktop implementation (pygame)
  mobile_adapter/  # Stub — ready for Kivy/BeeWare implementation
core/
  ports.py     # Abstract interfaces (InputPort, RendererPort, AudioPort)
  events.py    # Event bus
```

## Running on desktop

```bash
conda env create -f environment.yml
conda activate doodle_loot
python main.py
```

## Building the Android APK

Handled automatically by GitHub Actions on every push to `main`/`master`.

Go to **Actions** tab → latest run → **Artifacts** → download `DoodleLoot-debug-apk`.

First build takes ~30–45 min (downloads Android SDK + NDK). Subsequent builds use the cache and take ~5 min.
